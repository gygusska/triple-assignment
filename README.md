## 기술 스택

> React, TypeScript, Styled Components

### 선정 이유

* React
  * 기본요구사항

* TypeScript
  * 더 나은 개발 경험과 코드 퀄리티 향상
  * 타입 추론으로 에러를 방지

* Styled Components
  * 컴포넌트 단위 스타일을 하기 위해 사용
  * 자유로운 CSS 커스텀 컴포넌트를 만들 수 있어서 사용
  * 자사 스택

<br>

## 프로젝트 실행 방법

1. 레포지토리 클론

```
git clone https://github.com/gygusska/triple-assignment.git
```

2. dependencies 설치

```
npm install
```

3. 실행

```
npm run start
```
